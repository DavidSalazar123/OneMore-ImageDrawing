How to Install OneNote

Saturday, March 18, 2023

7:59 AM

Microsoft offers OneNote for free, whether or not you already have Office. If you already have Office, you can add OneNote without affecting your Office installation. If you don't have Office, you can install OneNote by itself.

 

📓

OneNote Desktop Edition

Microsoft has changed its OneNote strategy. Instead of promoting the OneNote for Windows 10 "app", they are continually integrating the best features from that app into the desktop edition of OneNote. The first iteratations of this were released in March 2023.

 

How to Install OneNote

Choose the appropriate installer to install OneNote by itself or in addition to an Office installation.

 

  • I have Office 64-bit or I have a 64-bit Windows machine
    • Download the OneNote 64-bit installer from here

 

  • I have Office 32-bit or I have a 32-bit Windows machine
    • Download the OneNote 32-bit installer from here

 

Click the appropriate link to download the standalone OneNote installer. It will be stored in your Downloads folder. You may need to right-click the file, open Properties, and tick the Unblock checkbox before continuing.

Which Version of Office Do I Have?

If you're not sure whether you have 32-bit Office or 64-bit Office, it's easy to fine out.

 

  1. Open Word
  2. Click File and the Account
  3. Click About Word
  4. A dialog window is displayed showing the full product version including whether it is 32-bit or 64-bit

Recommended Way to Install Office

If you already have Office installed, congratulations! You're good to go. Otherwise, I always recommend intalling 64-bit software if you have a 64-bit machine, which is highly likely if you purchased your machine in the last 15 years.

 

For some reason, the Office kit comes with a Setup.exe program that installs 32-bit Office, even if you run it on a 64-bit machine. But if you have a 64-bit machine, you can explicitly install 64-bit Office

 

  • Do not run the Setup.exe that comes with the Office kit
  • Instead, run Setup64.exe located under the Office subfolder instead - Office\Setup64.exe
