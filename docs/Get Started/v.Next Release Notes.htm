Feature Updates

  • Added reminder hints to the OneMore Calendar, showing a bell next to each page that contains one or more reminders
  • Updated Colorizer colors to be more compatible with light and dark mode. This lessens, but does eliminate, the need to recolor code when switching the page background between light and dark mode using the View/Switch Background command. Recoloring may still be required if a custom colors has been chosen for the page background.
  • Updated the Crop and Rotate window to enable the maximize button
  • Fixed an issue where styles were affecting more than the selected text, #1002
