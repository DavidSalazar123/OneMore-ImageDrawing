Online Services

Sunday, February 19, 2023

1:15 PM

OneMore uses the following online services within some of its commands. No information whatsoever is sent to these services beyond what is noted below. No personal information is shared. No information regarding your OneNote notebook structure or content is shared. No usage information is shared.

 

  • Edit/Pronunciate
    • Sends selected text to this service to convert to ruby text form
    • The Free Dictionary API - https://api.dictionaryapi.dev/api/v2/entries/{0}/{1}

 

  • Favorites/OneNote Keyboard Shortcuts
    • Downloads a Zip file containing a predefined .one page
    • https://github.com/stevencohn/OneMore/raw/main/Templates/OneNote_Keyboard_Shortcuts.zip

 

  • Images/Draw PlantUML
    • Sends selected PlantUML text to this service to convert to an image, see FAQ here

 

  • Search Engines
    • Any custom search engines that you specify in the Search Engines settings

 

  • About/Update
    • Discover and download new OneMore releases
    • https://api.github.com/repos/stevencohn/onemore/releases/latest

 

  • OneMore Installer
    • Download the WebView2 Evergreen Bootstrap installer
    • https://go.microsoft.com/fwlink/p/?LinkId=2124703
