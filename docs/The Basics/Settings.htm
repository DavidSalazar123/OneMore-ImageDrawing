Settings

11:14 AM

The Settings dialog customizes the behavior of some of some of the OneMore commands and experience.

 

General Options

Customizes the general behavior of OneMore.

 

Enable dynamic menu context settings

OneMore attempts to enable/disable its menu items based on the current focus. This can cause OneNote to become sluggish if commands are added to the Quick Access Toolbar. Disable this option if you've added OneMore command to the Quick Access Toolbar and you experience sluggish responsiveness.

 

Check for OneMore updates when OneNote starts

When OneNote starts, OneMore will silently check for updates by querying the OneMore Github pages. If there are no new updates, nothing will appear. If there is a new update, a dialog is displayed immediately after OneNote starts showing the new version and an option to update and restart OneNote.

 

Display Language

Sets the display language of OneMore menus and dialogs. This is separate from and does not affect the overall OneNote user interface and is available if you want to see a different language in OneNote than you have set on your workstation.

 

Colorize

The Colorize command will apply colorized syntax highlighted to the selected code block. By default, the font within the block is preserved and only the text color is affected. The Colorize options lets you set a preferred font family and size to apply when colorizing code blocks.

 

Command Aliases

The Command Palette is a powerful way to quickly invoke OneMore commands, but the names of some commands can be cumbersome even with the ability to filter on letters in the name. Command Aliases provides a way to create user-defined aliases for the built-in OneMore commands. These aliases are added to the command palette, in addition to the standard command names.

 

A recommended way to use this is to define short aliases for the custom styles, for example, Apply Style 1 ... Apply Style 6 can be aliased as H1 ... H6

 

Context Menu

Many of the OneMore commands can be added to the context menu that appears when you right-click on the body of a page.

 

Note that if you choose one of the "menu" options such as "Edit Menu" then that will be added as a fly-out menu and include all Edit Menu items. But if you choose one of the sub-items, such as "To lowercase" then that item will be added as a top-level context menu item. If you happen to select both then both will be shown, possibly duplicating items.

 

Note that the Proofing Language option is only available if you've configured Office with two or more proofing languages.

 

Favorites

Reorder, sort, or delete favorites. Pages are checked to confirm that they still exist and are reachable. If not, then they are highlighted and a tooltip will explain why.

 

 

Include reference to keyboard shortcuts page

If enabled then an item is added to the Favorites menu that links to a OneNote Keyboard Shortcuts page. The first time this favorite is clicked, OneMore will download and import the shortcuts reference page under the current section and navigate to that page. It may take a couple of seconds to appear. Subsequent clicks will simply navigate to that page again. This page can be moved to any open section or notebook and OneMore will still find it. If the page is deleted, clicking the Favorite link will download it again.

 

File Import

When importing PowerPoint and PDF files, each slide or page is rendered as an image before importing. The default image width is set to 600. The File Import options lets you set a preferred width when importing these file types.

 

Highlighter Themes

The rotating highlighter uses an array of colors to highlight selected text. Each time you select text and use this command, it will highlight the selected text using the next color in the array. The highlighter options let you choose which color theme to use.

 

Horizontal Lines

The single and double horizontal line snippets are simple text lines of hyphens and equal signs respectively. You can change the default color and length of these lines using this settings page.

 

Images

Customizes the behavior of image commands, including the default viewer used by the Open Image With command, the default resize width in the Resize and Adjust dialog and how PlantUML images are rendered.

 

Keyboard Shortcuts

A subset of OneMore commands are assigned default keyboard bindings. This page allows you to customize or clear those binding to suite your workflow or avoid conflicts with other installed addins and tools.

 

Navigator

The Navigator window and its underlying navigation service can be tuned. Change the maximum allowed length of the history list; when that limit is reached, the oldest item is removed from the list before the new one is added.

 

As you navigate from page to page, the navigation service will not record each page in the history list unless you spend "enough" time on that page. This time can be customized.

 

When you have one active screen, the Navigator window is restricted to the confined of that screen. When you have multiple screens set up in an extended configuration, you are able to move the Navigator window between them. You can restrict the Navigator window to the screen on which it was opened by setting the Restrict option.

 

Plugins

Lets you manage your custom plugins.

 

Quick Notes

Changes the behavior of the File Quick Notes command. See File Commands for more information.

 

Ribbon Bar

While all commands can be accessed from the More drop-down menu, some common commands can be added to the ribbon bar for quick single-click access.

 

Note that OneNote must be restarted if ribbon bar options are changed.

 

Search Engines

You can define one or more engines to be shown on the page context menu.

 

Each engine needs a unique name and a URL pattern. The URL pattern must contains the phrase "{0}" which will be replaced with a selected word or phrase. Some common search engine URL patterns are shown here:

 

Name

URL Pattern

Bing

https://www.bing.com/search?q={0}

Google

https://www.google.com/search?q={0}

Google Books

https://books.google.com/search?q={0}

Google Scholar

https://scholar.google.com/scholar?q={0}

Duck Duck Go

https://duckduckgo.com/?q={0}

Dictionary.com

https://www.dictionary.com/browse/{0}?s=t

WikipediA

https://en.wikipedia.org/wiki/{0}

 

If you define one engine then that is shown directly in the page context menu. If you define more than one engine, then a submenu is added to the context menu with a list of engines that you define.

 

Snippets

Snippets are automatically ordered alphabetically but you can delete individual snippets using the Snippets Manager
