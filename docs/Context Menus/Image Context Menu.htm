Right-click an image to display its context menu. OneMore commands appear at the top of this menu as shown here.

 

 

Add Caption

See also Image Commands

Adds a caption to the selected image.

 

Crop and Rotate

See also Image Commands

Opens an image editing window that allows you to precisely crop an image

 

Open Image With

See also Image Commands

Opens the selected image in an external image viewer/editor.

 

Resize and Adjust

See also Image Commands

Resize and adjusts images on the page.
