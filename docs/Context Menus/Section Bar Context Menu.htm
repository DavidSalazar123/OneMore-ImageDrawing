Section Bar Context Menu

Sunday, February 19, 2023

10:20 PM

Right-click an empty spot in the section tabs bar to display its context menu. OneMore commands appear at the top of this menu as shown here.

 

 

Archive Notebook

See also File Commands

Creates a zip file of all sections in the current notebook.

 

Number Sections

See also Numbering Commands

Adds sequential prefixes to all sections in the current notebook.

 

Remove Section Numbers

See also Numbering Commands

Removes the numerical prefixes from all sections in the current notebook that were previously added using the Number Sections command.

 

Sort Sections in Notebook

Sorts the sections in the current notebook.
